Employees Resource

This resource provides access to employee data.

GET api/employees

Returns a list of all employees.

Headers:

  • Accept: application/json
  • Accept: application/xml

Response:

Returns a JSON array or XML document of employee objects.



GET api/employees/{id}

Returns a list of all employees.

Headers:

  • Accept: application/json
  • Accept: application/xml

Path Parameters:

  • id (integer): The ID of the employee

Response:

Returns a JSON object or XML document of the specific employee.



POST api/employees

This resource to create an employee record in the database.

Headers:

  • Content-Type: application/json
  • Content-Type: application/xml

Response:

"Employee added successfully" 201 Created



PUT api/employees/{id}

to edit specific employee with {id} .

Headers:

  • Content-Type: application/json
  • Content-Type: application/xml

Response:

"Employee updated successfully" 200 OK

Path Parameters:

  • id (integer): The ID of the employee




DELETE api/employees/{id}

This resource to delete specific employee .

Response:

"Employee deleted successfully" 200 OK
